## 功能

- 使用简单
- 丰富的功能
- 语义明确

## 开发

```bash
git clone git@github.com:fujia-dev/hammer.git

cd hammer

npm install
```

## 安装

```bash
npm i @fujia/hammer

# or use yarn
yarn add @fujia/hammer
```

## 使用

你可以将整个包导入来使用：

```javascript
import _ from '@fujia/hammer'
```

除此之外，也可以引入单个文件来使用:

```javascript
import throttle from '@fujia/hammer/dist/throttle'
```
